This research proposes a novel hybrid deep learning and gradient-boosted tree framework engineered specifically for IoT edge deployments—environments where computational budgets are tight and adversarial threats are accelerating in scale and sophistication.

The architecture fuses a Deep Neural Network (DNN) that learns complex non-linear feature representations from raw network traffic with XGBoost for robust ensemble classification, achieving detection fidelity well above 97% while operating within stringent edge hardware constraints.

A defining contribution is full Explainable AI (XAI) integration via SHAP and LIME, enabling security analysts to understand and audit every model decision—addressing the "black box" problem that has historically blocked ML adoption in safety-critical security infrastructure.

Validated against CIC-IoT2023 and TON_IoT benchmark datasets, the framework consistently outperforms single-model baselines while maintaining real-time performance on resource-constrained hardware running at under 180MB RAM and 15% CPU overhead.
